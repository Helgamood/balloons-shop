# Balloons Shop 🎈

Учебный проект интернет-магазина на чистом JavaScript с использованием Vite.

## О проекте

Проект представляет собой простой интернет-магазин воздушных шаров.

Цель проекта - практика работы с интерфейсом, логикой приложения и структурированием кода.

## Технологии

- HTML
- CSS
- JavaScript (Vanilla)
- Vite
- Git
- БЭМ

## Функционал

- Отображение списка товаров
- Добавление товаров в корзину
- Подсчёт общей стоимости
- Базовая структура компонентов на JS
